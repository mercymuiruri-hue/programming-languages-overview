## Python
*Contributed by: Mercy Muiruri*

### Overview
Python is a high-level, interpreted programming language known for its exceptional readability and clean syntax. Created by Guido van Rossum and released in 1991, Python was designed with a philosophy that emphasizes code readability, allowing developers to express complex concepts in fewer lines of code than languages like C++ 

### Key Features
* **Simple & Readable Syntax:** Python uses clean, English-like syntax and relies on clean indentation rather than curly braces `{}`  making it highly accessible for beginners.
* **Interpreted Language:** Code is executed line-by-line, which simplifies debugging and eliminates the need for a separate compilation step before running programs.
* **Dynamically Typed:** You do not need to explicitly declare variable types (like `int` or `string`) when writing code; Python determines the type automatically at runtime.
* **Extensive Standard Library:** Known as a "batteries included" language, 

### Common Use Cases
* **Data Science & Machine Learning:** Python is the industry standard for AI, data analysis, and predictive modeling.
* **Web Development:** Powering the back-end of massive platforms (like Instagram and Spotify) using robust web frameworks like Django, Flask, and FastAPI.
* **Automation & Scripting:** Ideal for writing quick scripts to automate repetitive tasks, scrape web data (using BeautifulSoup), or manage system files.
